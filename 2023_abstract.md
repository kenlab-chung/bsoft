# 2023 Abstract
## 1、 2023
 1. 车载医保结算终端

    - 在2022年基础上增加了1/3以上功能。
    - 新增内容：补退费；补欠费；药品拆零； 疾病类型、药品、治疗项目等新增常用便签；一车多人车费计算；电子发票。
 
 2. 软交换
    
    - 国产化：麒麟v10 arm 平台，达梦数据库
    - 容器化：docker云部署
    - 新增特色功能：5G音视频通信开发。
      
 3. 音视频服务程序
    
    - 解决车载端视频在移动端、web端显示问题（Websocket协议）。
    - 支持GB28181、JT1078、RTSP等协议。目前已接入设备：海康威视摄像终端，大华视频终端。
    - 容器化：docker云部署
    
 4. 旧版C++调度系统
